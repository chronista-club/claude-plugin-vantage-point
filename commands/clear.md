# Clear Dashboard

Clear content from Vantage Point dashboard panes.

## Usage

Use the `mcp__vantage-point__clear` tool to clear pane content.

Optionally specify `pane_id` to clear a specific pane:
- "main" - Clear main pane
- "left" - Clear left pane
- "right" - Clear right pane

Without pane_id, clears all panes.
